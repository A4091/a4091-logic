# A4091 logic

This project contains the GAL logic used on the A4091 board.

## Original Timings

The original chips use the following timings

|  ID  |   Type   |
|------|----------|
| U304 | 22V10-10 |
| U207 | 22V10-15 |
| U205 | 22V10-15 |
| U203 | 22V10-15 |
| U306 | 22V10-10 |
| U202 | 22V10-15 |
| U305 | 22V10-10 |
| U303 | 22V10-15 |
